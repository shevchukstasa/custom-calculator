# Запуск калькулятора печи

## Всегда используйте эту ссылку:

```
http://localhost:5173
```

## Как запустить:

### Вариант 1: Через скрипт (рекомендуется)
```bash
./start.sh
```

### Вариант 2: Вручную
```bash
npm run dev
```

## Если порт занят:

Остановите старые серверы:
```bash
lsof -ti:5173 | xargs kill -9
```

Затем запустите снова:
```bash
npm run dev
```

## Постоянная ссылка:

⭐ **http://localhost:5173** ⭐

Эта ссылка будет работать всегда, если сервер запущен.

## Проверить, запущен ли сервер:

```bash
lsof -i:5173
```

Если ничего не показывает — сервер не запущен.
