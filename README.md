# Калькулятор печи - Moonjar Design

## Быстрый старт

### Запуск сервера:
```bash
./start.sh
```

Откройте: **http://localhost:5173**

## Разделы

### 1. Загрузка печи
- Размеры изделия
- Тип изделия (плитка, столешница, раковина, 3D)
- Форма (квадратная/прямоугольная, круглая, треугольная)
- Расположение глазури
- Автоматический расчет для обеих печей
- Результаты: количество, площадь, метод загрузки

### 2. Расчет стоимости
- Выбор печи
- Цена камня
- База данных камней
- Автоподбор камня
- Два режима: простой и расширенный
- Результаты для Indonesia и Abroad
- Сравнение рынков

## Файлы

### Основные:
- `src/App.tsx` - главный компонент
- `src/App.css` - стили Moonjar
- `src/utils/kilnCalculations.ts` - расчеты загрузки
- `src/utils/costCalculations.ts` - расчеты стоимости

### Компоненты:
- `src/components/ProductInput.tsx` - ввод размеров
- `src/components/ProductTypeSelector.tsx` - выбор типа
- `src/components/GlazePlacementSelector.tsx` - выбор глазури
- `src/components/MultiKilnResults.tsx` - результаты печей
- `src/components/CostResults.tsx` - результаты стоимости
- `src/components/CostParameters.tsx` - параметры расчета
- `src/components/ModeToggle.tsx` - переключатель режима
- `src/components/StoneDatabase.tsx` - база камней
