# Инструкция по запуску

## Быстрый старт

1. Откройте терминал в папке `kiln-calculator`
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите приложение:
   ```bash
   npm run dev
   ```
4. Откройте в браузере: `http://localhost:5173`

## Если npm install не работает

Если у вас проблемы с сетью, попробуйте:
```bash
npm install --registry=https://registry.npmjs.org/
```

Или используйте yarn:
```bash
yarn install
yarn dev
```

## Структура приложения

- `/src/components/` - React компоненты интерфейса
- `/src/utils/` - Логика расчетов
- `/src/types/` - TypeScript типы
- `/src/App.tsx` - Главное приложение
- `/src/App.css` - Стили

## Тестирование расчетов

Для тестирования логики расчетов без запуска приложения:
```bash
node test-calc.js
node test-detailed.js
node test-edge.js
```
